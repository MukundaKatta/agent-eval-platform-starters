## Project Title

Agent Evaluation Field Notes

## Short Description

Reusable scorecards, replay checklists, and RAG guardrail smoke tests for evaluating tool-using AI agents.

## One-Line Bio

Mukunda Rao Katta is an AI/ML engineer building practical agent systems, evaluation workflows, developer tools, and public research artifacts.

## About The Project

Agent Evaluation Field Notes is a compact collection of practical evaluation artifacts for tool-using AI agents. It includes operational scorecard templates, trajectory replay debugging checklists, and RAG guardrail smoke tests for prompt injection and vector poisoning. The project is designed for builders who need repeatable evaluation workflows that are small enough to maintain and concrete enough to use in production reviews.

## What It Contains

- Operational scorecard template for tool-using agents
- Trajectory replay debugging checklist
- RAG guardrail smoke tests
- Public links to related preprints, demos, notebooks, and source repos
