## Reusable Short Description

Agent Evaluation Field Notes is a compact public collection of scorecard templates, trajectory replay checklists, and RAG guardrail smoke tests for evaluating tool-using AI agents. It is designed for builders who need practical, repeatable evaluation workflows rather than a large benchmark.

## Reusable Long Description

Agent Evaluation Field Notes collects reusable templates and small public artifacts for evaluating tool-using AI agents. The materials cover operational scorecards, trajectory replay debugging, prompt-injection smoke tests, vector-poisoning checks, and rollout-readiness criteria. The project is intentionally lightweight: each artifact is small enough to copy into a real engineering workflow and concrete enough to support repeatable review.

The notes connect to public research artifacts, demos, and datasets around lightweight agent evaluation, including public scorecard apps, notebooks, preprints, and reusable snippets.
