# Dataset Metadata

Title: Agent Evaluation Field Notes Scenario Set

Author: Mukunda Rao Katta

Description: A compact scenario set for testing operational scorecards, trajectory replay, and guardrail checks for tool-using AI agents. The dataset is intentionally small and designed for repeatable evaluation workflows rather than large-scale benchmarking.

Keywords: AI agents, agent evaluation, LLM evals, scorecards, trajectory replay, RAG guardrails, prompt injection
